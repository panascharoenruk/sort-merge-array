# Merge Sorted Arrays

TypeScript project for merging 3 sorted arrays without using any built-in sort function.

## Requirements

- Node.js >= 18
- npm >= 9

---

## Installation

Clone repository:

```bash
git clone <your-repo-url>
cd merge-sorted-array
```

Install dependencies:

```bash
npm install
```

---

## Run Unit Tests

```bash
npm test
```

Watch mode:

```bash
npm run test:watch
```

---

## Build Project

```bash
npm run build
```

Compiled files will be generated in:

```bash
dist/
```
