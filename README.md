# Merge Sorted Arrays

Merge and sorted 3 arrays into one with O(n) complexity

## Requirements

- Node.js >= 18
- npm >= 9

---

## Installation

Clone repository:

```bash
git https://github.com/panascharoenruk/sort-merge-array.git
cd sort-merge-array
```

Install dependencies:

```bash
npm install
```

---

## Run Unit Tests

```bash
npm test
```
---

## Build Project

```bash
npm run build
```

Compiled files will be generated in:

```bash
dist/
```
